
# Command Line 基礎編

> コマンドラインはコンピュータにプログラムを実行させる命令文  
>> 文章だけでコンピュータを操作することができる。  
SQLが文章だけでDBを操作できるのと同じ。  
今回学ぶのは基本、macやunixで使えるもの。

## touch ファイル名
- 空のファイルを作成することができる。

## cat ファイル名

- ファイルの中身を表示することができる。

## 補完機能
- 入力途中で「tabキー」入力で最後まで打ってくれる。

## mkdir ディレクトリ名
- ディレクトリを作ることができる

## cd ディレクトリ名
- ディレクトリの移動ができる
